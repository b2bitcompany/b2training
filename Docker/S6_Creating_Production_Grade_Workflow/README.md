# b2training - Docker
**Developed by [b2bit.company](https://b2bit.company)**

## 🚀 Getting Started

This project includes a pre-generated React boilerplate as part of the **b2training - Docker** course. Follow the steps below to set up your local development environment.

### Prerequisites

1.  **Download** the provided boilerplate ZIP file.
2.  **Unzip** the contents to your preferred development directory.

### Installation

Navigate into the frontend directory and install the necessary dependencies:

```bash
cd frontend
npm install
```

---

### ⚠️ Important Note on `npm audit`

**Do not run `npm audit fix` or `npm audit fix -f`.**

Running these commands may update dependencies to versions that are incompatible with this boilerplate, potentially breaking the project. 

* **Warnings are not errors:** You can safely ignore security warnings during the installation process.
* **Context:** For a deeper dive into why `npm audit` can be problematic for React projects, see this article: [npm audit: broken by design](https://overreacted.io/npm-audit-broken-by-design/).
